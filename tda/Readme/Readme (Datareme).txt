スキンシェーダ 2013年01月24日版

ニコニコ動画にアップロードした動画「【MMD】皮膚の質感について考えてみました(第2回)」
において使用した、MMEエフェクトファイルと法線マップの詰め合わせです。

ビームマンP様の「多光源エフェクト」を改造したものとなります。
http://www.nicovideo.jp/watch/sm19312025

●2013年01月24日版の修正箇所

・両面描画（α=0.99）をおこなう材質において、裏面が描画する際に、リムライトが最大強度で描画されていたのを修正。
・半透明の材質の場合、視線方向と材質の法線の向きと考慮して透明度を求めるよう修正。
・SphereMapScale(スフィアマップの色を強調する係数)の値を3から1に変更
・各種パラメータの調整。

補足　修正前のエフェクトファイルにつきましては「20130118版」フォルダ内に格納してあります。

●各エフェクトについて
「pmotskin.fx」
人間の皮膚の質感を表現するエフェクトファイルです。
とはいうものの、蝋や樹脂のように見えます。
dummy_n.bmp、skin2_normal.bmpという法線マップを使用しています。

以下のエフェクトファイルは、処理自体はpmotskin.fxと同じです。
法線マップとパラメータが異なるだけです。

「pmotskin_つや.fx」
眼球等のつやのある箇所に使用したエフェクトファイルです。

「pmotskin_布」
衣装に使用したエフェクトファイルです。
nuno_n.bmpという法線マップを使用しています。

「pmotskin_髪横向き」
頭髪に使用したエフェクトファイルです。
hair_h_n.bmpという法線マップを使用しています。

「pmotskin_髪縦向き」
頭髪に使用したエフェクトファイルです。
hair_v_n.bmpという法線マップを使用しています。
