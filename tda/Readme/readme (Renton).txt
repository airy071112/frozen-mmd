■使い方
------------

モデル改造

モデルの改造は『NCHLShader2』のカスタマイズ指南.txtを参照してください。
改造のさいに、PMMXEの材質で全てのスフィアを削除してください。
ここで布や革などの材質にしたいものは、ｽﾌｨｱの欄でサブTexを選択し、Normalmapフォルダ内のPNGファイルを読み込んでください。
モデルのUV展開によっては歪むこともあるので、PMXEのUVエディタプラグインなどで調整するといいかもです。
※Skinは肌というよりちょっとザラついた感じかもしれません。好みで使いたいところで使ってください。


MMDで使う

MMDでは『NCHLShader2』の説明書.txtを参照して導入してください。
導入が済んだら、改造モデルを読み込んでください。
MMEを開いてエフェクト割り当てウィンドウのMainタブで、改造したモデルをサブセット展開し、ファイル選択で“RT_Fabric01.fx”などを材質に合わせて選択してください。
好みに合わせて調整したり、『NCHLShader2』のプリセットなども試してカスタマイズしてください。
