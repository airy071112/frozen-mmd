以下は使い方などになります。

読み込む順

※注意
以下の説明でモーションはすべてD&Dで読み込んでますが、D&DでMMDが強制終了する場合は 上のメニューのファイル→モーションデータ読込からファイルを指定して読み込んでください。

まずおかっちさんのモーションをDLし、
http://www.nicovideo.jp/watch/sm24059084
音源のwavファイルと好きなモデルを用意してください。

MMDを起動し、pmxフォルダ内のpmxを01から順にD&Dします。10_crystal.pmxの次に 用意した好きなモデルをD&Dし、モデルの後に11_stair.pmxをD&Dします。 （10と11の間にモデルを入れる）

x_MMEフォルダ内の4つのxファイルをこの順にD&Dします。
1 ground1.x
2 ground2.x
3 AutoLuminous.x
4 wf_c4.x

カメラ編ボタンをクリックして画面を切り替え、アクセサリ操作パネルで以下の3つのxファイルの 追従ボーンを指定します。
ground1.x → 02配置用ダミ、ground1
ground2.x → 02配置用ダミ、ground2
wf_c4.x → 06城、for_wf_c4

アクセサリ操作パネルでAutoLuminous.xのSiを0.2に、wf_c4.xのTrを0.4に書き換えます。

上のメニューの背景→アクセサリ編集をクリックし、1を2に書き換えます。

vmdフォルダ内の01から11のvmdを同じ番号のpmxにD&Dします。

おかっちさんのLet it go-motion.vmdを用意した好きなモデルにD&Dします。

カメラ編ボタンをクリックして画面を切り替え、おかっちさんのLet it go-camera.vmdをD&Dし、 さらにvmdフォルダ内のAdditionalCamera.vmdをD&Dします。

照明操作パネルの数値を80、80、90に書き換えて登録をクリック、座標軸ボタンをクリックしてOFFに、 上のメニューの表示→地面影表示のチェックをはずしてOFFにします。

wavファイルを読み込んで再生！

その他
・城とバルコニーの床にはマントを付けた時の床判定用に剛体を入れてあります。ボーンで剛体のみを 移動できるので邪魔なときは下に動かすなどしてください。
・ドアの剛体は、最後のドアが閉まるシーンはモデルとドアがかなり近いので髪などのはみだし防止用に 入れています。
・城の数字パネルは柱OFFなどのモーフを自分で動かす時の確認用です。
・スカイドームは大きさの違う3つの球が重なっており、頂点モーフのONOFFで3つのテクスチャが 切り替わるようになっています。
・モーションやカメラに合うよう階段や床・空が動く部分があるのでカメラを変えるときはご注意ください。
・ground1.xとground2.xをダミーボーンにつけてるのは地下に潜らせてOFFの代用にしているためです。 指定のフレームでOFFにするよりvmdを読み込む方が簡単かなと思ってそうしましたが かえってややこしかったかもしれません・・
・EnglishLip.vmdはなんとなく英語で歌ってる感じのするリップです。使う場合は 元のあいうえおを全て削除してから読み込んでください。階段をつくるあたりの数小節は あえて歌っていません。
・最後のドアが閉まるところでカメラの中心とドアの中心が合っていません。ドアは独立してますので 気になる方は05_door.pmxの全ての親で調節してください。
